# Sales Analytics for Online Store  
**SQL · Python · Google BigQuery · Tableau**

## Опис проєкту
Аналітичний проєкт з аналізу продажів інтернет-магазину, що охоплює повний цикл роботи з даними: від формування датасету в SQL до статистичного аналізу та побудови дашборду.

## Інструменти
- SQL (Google BigQuery)
- Python (Jupyter / Google Colab)
- Tableau Public

## Основні кроки
- формування аналітичного датасету з кількох таблиць у BigQuery
- розвідувальний аналіз та візуалізації в Python
- аналіз динаміки продажів і сезонності
- статистичний аналіз взаємозв’язків і відмінностей між групами
- створення аналітичного дашборду в Tableau

## Ключові напрямки аналізу
- продажі за країнами, континентами та категоріями товарів
- структура продажів за девайсами та каналами трафіку
- поведінка зареєстрованих і незареєстрованих користувачів
- кореляції та статистично значущі відмінності між групами

## Результат
Отримано цілісний аналітичний звіт і дашборд, що дозволяє оцінити структуру та динаміку продажів інтернет-магазину.
